# عطر مال — پیش ثبت نام روز عطر مال ۱۴۰۵

**وب‌سایت رسمی فروشگاه عطر مال** برای پیش ثبت‌نام رویداد **روز عطر مال ۱۴۰۵** با تخفیف ویژه ۲۵٪.

**دامنه:** [atrmal.ir](https://atrmal.ir)

---

## ویژگی‌ها

- **فرم ثبت‌نام:** نام و نام خانوادگی، ۴ رقم آخر شماره ملی، شماره تماس (اختیاری)
- **تشخیص ثبت‌نام تکراری:** بر اساس نام (حذف فاصله‌ها و نرمال‌سازی)
- **تبدیل خودکار اعداد فارسی/عربی به انگلیسی** در فیلد کد ملی
- **تایمر شمارش معکوس** تا تاریخ ۱۴۰۵/۰۵/۰۵
- **بنر سفارشی:** پس از ثبت موفق، نام و کد کاربر روی بنر اول نمایش داده می‌شود
- **دانلود بنر با کیفیت بالا:** با html2canvas (scale: 6)
- **ذرات طلایی شناور (✦):** ۷۵ عدد در تمام صفحه
- **پنل مدیریت:** جستجو، ویرایش/حذف تکی و گروهی، آمار، خروجی CSV
- **نمایش ۵ شعبه** با لینک نقشه نشان
- **طراحی واکنش‌گرا (Responsive)**
- **قابلیت fallback به localStorage** در صورت عدم دسترسی به سرور

## نصب و راه‌اندازی

### هاست اشتراکی (cPanel / DirectAdmin)

1. تمام فایل‌های پروژه را در **public_html** آپلود کنید.
2. اطمینان حاصل کنید پوشه `data/` قابل نوشتن باشد (permission 755 یا 775).
3. مسیر `admin.html` برای دسترسی به پنل مدیریت.
4. پسورد پیش‌فرض پنل: `atrmal@1405`

### لوکال (XAMPP / WAMP)

```bash
# کلون کردن پروژه
git clone https://github.com/your-username/atrmal.git

# انتقال به پوشه htdocs
cp -r atrmal /path/to/htdocs/

# دسترسی نوشتن برای پوشه data
chmod 755 atrmal/data/
```

سپس در مرورگر باز کنید:
```
http://localhost/atrmal/
```

### قابلیت‌ها

- **آمار:** تعداد کل ثبت‌نام‌ها، ثبت‌نام‌های امروز، تعداد همراه با شماره تماس
- **جستجو:** بر اساس نام، کد ملی، شماره تماس
- **ویرایش تکی:** کلیک روی دکمه ویرایش در هر سطر
- **حذف تکی:** کلیک روی دکمه حذف در هر سطر
- **عملیات گروهی:** انتخاب چند کاربر با چک‌باکس → ویرایش یا حذف گروهی
- **خروجی CSV:** دانلود تمام کاربران در قالب CSV
- **بروزرسانی (Refresh):** دریافت آخرین اطلاعات از سرور
- **پاک کردن اطلاعات:** حذف تمام داده‌های سرور و مرورگر
- **خروج:** بازگشت به صفحه ورود

---

## نکات امنیتی

- پنل مدیریت فقط با نشست (session) معتبر قابل دسترسی است.
- فایل `data/users.json` با `.htaccess` از دسترسی مستقیم محافظت می‌شود.
- در صورت عدم دسترسی به سرور، برنامه به `localStorage` مرورگر fallback می‌دهد.
- پسورد پیش‌فرض را پس از نصب تغییر دهید.
